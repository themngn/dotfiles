# mono's dotfiles

## Clone repo
```
git clone https://github.com/themngn/dotfiles.git .dotfiles
```
or using SSH
```
git clone git@github.com:themngn/dotfiles.git .dotfiles
```
## Run install script (automatic)

```
sudo ./0install.sh
```

## Login into Hyprland and run hyprland.sh (Not automatic)

```
./1hyprland.sh
```
